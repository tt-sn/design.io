───Study:

研究室で行っている研究の紹介です。
タイトル：人体動作に連動した3D水流ビジュアルエフェクトの生成

 / 水の見た目の生成 / 

UnityのShaderを使って、水を表現した作品になります。画像では、球のカメラ方向の逆方向にある背景をテクスチャとして取得し、球の頂点の法線ベクトルに則って歪ませることで水の屈折を再現しました。

/ 自由位置で水を生成 /

Unity内の３次元座標の任意の位置*から自由に水を生成することにより、非現実なエフェクトとして用いることができます。(*:MC法というボリューム空間を区切る軽量化の手法を用いてるので、範囲の制約があります。)

 / 水の形状の生成 / 

水や雲など、CGにおける流体の形状の表現に用いられる、メタボールという”滑らかな曲線”を生成する手法を用いて、Unity内で動的に”水っぽい物体”を生成できるようにしました。

 / 身体から水を生成 / 

Kinectを用いて、人体の部位の座標を取得。そして加速度を計算することによって動きに合わせて飛んだりする水を生成することができます。 主にゲームやCGを用いたライブステージなどでの利用が期待できます。
